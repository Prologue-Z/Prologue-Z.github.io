阿里云 ECS SSH 快速配置指南

ZhangXu
2026-03-09

💡 核心发现：阿里云控制台可以直接生成密钥对并绑定实例，无需手动生成 SSH 密钥！网上大部分教程都在教复杂的 ssh-keygen 流程，其实根本不需要。

🚀 正确做法（3 步搞定）

步骤 1：在阿里云控制台创建密钥对

  1. 登录 阿里云控制台
  2. 进入 网络与安全 → 密钥对
  3. 点击 创建密钥对
  4. 下载私钥文件

步骤 2：绑定密钥对到 ECS 实例

  1. 进入 ECS 实例列表
  2. 选择目标实例
  3. 点击 更多 → 密码/密钥 → 绑定密钥对
  4. 选择刚才创建的密钥对
