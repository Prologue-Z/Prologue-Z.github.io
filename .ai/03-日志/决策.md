## 2026-03-19 决策记录

### 决策 1: 部署方式
**时间**: 22:50  
**决策**: 使用 `hexo deploy` 一键部署  
**原因**: 
- 官方推荐方式
- 简单易用
- 适合个人项目

**配置**:
```yaml
deploy:
  type: git
  repo: git@github.com:Prologue-Z/Prologue-Z.github.io.git
  branch: gh-pages
```
