## 工作流程

### 1. 内容创建流程
```
用户提供素材 → 内容专家撰写 → Reviewer 审查 → Leader 确认 → 发布
```

### 2. 主题优化流程
```
用户反馈 → 前端专家调整 → Reviewer 测试 → Leader 确认 → 部署
```

### 3. 媒体处理流程
```
用户提供素材 → 媒体专家处理 → Reviewer 审查 → Leader 确认 → 上传
```

## 沟通机制

### 阶段确认
每个阶段结束必须找用户确认后才能继续：
- ✅ 阶段 1: 基础框架（已完成）
- ⏳ 阶段 2: 内容填充（进行中）
- ⏳ 阶段 3: 主题优化
- ⏳ 阶段 4: 最终审查

### 实时记录
- 重要决策立即写入 `03-日志/决策.md`
- 讨论内容写入 `03-日志/讨论/`
- 版本更新写入 `00-入口/版本.md`
